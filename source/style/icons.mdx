The Fuse Design System provides a consolidated set of iconography for use throughout NI software products.

## XAML Code Snippet
To reference an icon from this library in XAML, use the pack URI scheme of the icon.

```xml
<Image Source="pack://application:,,,/NationalInstruments.PlatformFramework;component/ProjectExplorer/Images/AddPullDown_16x16.png"/>
```

import iconGroups from './icons.json'

<link rel='stylesheet' href='../../icons/icons.css'/>
<ul class="remove-bullet">
{ iconGroups.map(group => (
  <li> { group.path } 
    <div class="flex-container">{ group.icons.map(icons => (
      <div class="flex-item flex-tile"> 
        <div class="flex-tile-item"><img class="icon" src={ '../../icons/' + group.path + '/' + icons.icon }/> </div>
        <div class="flex-tile-item">{ icons.icon }</div>
      </div>
    ))}
    </div>
  </li>
  ))}
</ul>
