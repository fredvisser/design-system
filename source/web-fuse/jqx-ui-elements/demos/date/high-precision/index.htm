High-precision DateTime object example

Initialization:

new JQX.Utilities.DateTime(1920, 3, 12, 8, 34, 21, 89, 19, 800, 2, 333, 18, 45, 11);

Alternative initialization:

new JQX.Utilities.DateTime('1920-03-12 08:34:21:089:019:800:002:333:018:045:011');

format 'FT' - full precision time (from hours to yoctoseconds (10−24 seconds)):
