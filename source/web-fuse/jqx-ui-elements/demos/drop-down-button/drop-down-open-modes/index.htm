Modes Description

  • Auto - the dropDownButton is opened/closed when the element is hovered.
  • Default - the dropDownButton is opened/closed when the element is clicked.
  • DropDownButton - the dropDownButton is devided in two components. An Action Button and a dropDown Button. The dropDownButton can be opened/closed when the dropDownButton is clicked.
  • None - the dropDownButton can't be opened/closed.
