jqxPager

jqxPager provides navigation for paged information, typically used with a grid or table.

Contents

  • Usage
  • Style
  • Mobile

Usage

Pager with default settings allows the user to navigate between 3 items. This range can be changed by setting:

  • items : total number of items
  • itemsPerPage : the number of items displayed on single page

Navigation

User can navigate between pages and/or items via built-in optional buttons.
